# Sentinel AI — API Reference

Base URL: `/api/v1` · Auth: `Authorization: Bearer <JWT>` (from `/auth/login`)
Interactive docs: `GET /docs` (Swagger UI, auto-generated by FastAPI).

## Auth
| Method | Path | Role | Description |
|---|---|---|---|
| POST | `/auth/login` | — | `{username, password}` → `{access_token, role}` |

Demo users: `admin/admin123`, `commander/command123`, `responder/respond123`,
`hospital/hospital123`, `citizen/citizen123`.

## Incidents
| Method | Path | Role | Description |
|---|---|---|---|
| GET | `/incidents` | any | List incidents (Firestore) |
| POST | `/incidents` | any | Create manual incident |
| POST | `/incidents/report/voice` | any | Audio → STT → Gemini → structured incident |
| POST | `/incidents/report/image` | any | Image → Vision AI + Gemini hazard analysis |
| PATCH | `/incidents/{id}/status?status=` | responder+ | Update status |

## Predictions (all responses include an `explanation` XAI block)
| Method | Path | Description |
|---|---|---|
| POST | `/predictions/flood` | `{rain_6h_mm, river_level_m, drainage_capacity, soil_saturation, elevation_m}` |
| POST | `/predictions/accident` | `{congestion, rain_mm, hour, is_night, road_quality}` |
| POST | `/predictions/fire` | `{temp_c, humidity, wind_kmh, building_age, industrial}` |
| GET | `/predictions/hospital-occupancy?hospital=&days=` | 7-day occupancy forecast |
| GET | `/predictions/ambulance-demand?district=&hours=` | Hourly demand forecast |
| GET | `/predictions/traffic-congestion?district=&hours=` | Congestion forecast |

## AI & Agents
| Method | Path | Role | Description |
|---|---|---|---|
| POST | `/chat` | any | `{message, mode?}` — auto-routes to RAG / decision / general Gemini |
| POST | `/agents/decision-cycle` | responder+ | Full multi-agent fusion (also fired by Cloud Scheduler) |
| GET | `/agents/runs` | any | Agent execution audit |
| GET | `/decisions` | any | Fused decisions with reasoning |
| POST | `/rag/ingest` | commander+ | Upload SOP PDF → Document AI → Vector Search |
| POST | `/routes/emergency` | any | Hazard-aware routing (Google Maps) |

## Dashboard
| Method | Path | Role | Description |
|---|---|---|---|
| GET | `/dashboard/summary` | any | Home KPIs + latest risk |
| GET | `/dashboard/map` | any | Map layers: incidents/hospitals/closures/flood |
| GET | `/dashboard/timeline` | any | Merged event timeline |
| GET | `/dashboard/hospitals` `/weather` `/traffic` | any | Domain dashboards |
| GET | `/dashboard/admin/audit` | admin | Notifications, work orders, reports, agent runs |

## Error model
`401` invalid/expired token · `403` insufficient role · `422` validation error
(FastAPI detail array) · `500` logged to Cloud Logging with request context.
